Git Terminology

When working with Git and the exercises in this Learning Lab, be aware of the following terminology.

  • Repository (Repo): A vault for storing version-controlled files.
  • Working directory: The visible directory and its contents.
  • Versioned files: Files that you have asked Git to track.
  • Un-versioned files: Files in your working directory that are not tracked by Git.
  • Commit: A snapshot in time of your version-controlled files.
  • Branch: A safe place for you to work.
